### 4. إصلاح عرض عنوان The Godfather في صفحة المشاهدة
**الحالة:** ✅ تم الحل

**المشكلة:**
- فيلم The Godfather يظهر اسمه بالعربية "العراب" في صفحة `/movies/trending`
- لكن يظهر بالإنجليزية "The Godfather" في صفحة `/watch/movie/the-godfather`
- قاعدة البيانات تحتوي على العنوان العربي الصحيح: `title_ar = "العراب"`
- API يعيد البيانات الصحيحة مع حقل `title_ar`

**السبب الجذري:**
- مكون Watch كان يستخدم `dualTitles.sub` (العنوان الثانوي) بدلاً من `dualTitles.main` (العنوان الرئيسي)
- عندما تكون اللغة عربية:
  - `dualTitles.main` = العنوان العربي (العراب)
  - `dualTitles.sub` = العنوان الإنجليزي (The Godfather)

**الحل:**
- تغيير السطر 353-356 في `src/pages/media/Watch.tsx`:
```typescript
// قبل الإصلاح (خطأ)
const title = useMemo(() => {
  if (dualTitles.sub) return dualTitles.sub  // ❌ يعيد الإنجليزي
  return details?.title || details?.name || (type === 'movie' ? `فيلم #${id}` : `مسلسل #${id}`)
}, [details, type, id, dualTitles])

// بعد الإصلاح (صحيح)
const title = useMemo(() => {
  if (dualTitles.main) return dualTitles.main  // ✅ يعيد العربي
  return details?.title || details?.name || (type === 'movie' ? `فيلم #${id}` : `مسلسل #${id}`)
}, [details, type, id, dualTitles])
```

**الملفات المعدلة:**
- `src/pages/media/Watch.tsx` (السطر 353-356)

**التحقق:**
- الآن صفحة `/watch/movie/the-godfather` تعرض "العراب" بشكل صحيح
- يطابق السلوك في صفحة `/movies/trending`
