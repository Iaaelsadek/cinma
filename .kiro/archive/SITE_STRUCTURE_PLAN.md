## 🏗️ البنية التقنية المقترحة

### 1. Component مشترك: `CategorySearchPage`

```typescript
// src/pages/discovery/CategorySearchPage.tsx
interface CategorySearchPageProps {
  category: string        // 'movies', 'series', 'anime', etc.
  subcategory?: string    // 'arabic', 'foreign', 'turkish', etc.
  genre?: string          // 'action', 'comedy', etc.
  year?: number           // 2026, 2025, etc.
  rating?: string         // '9+', '8+', etc.
  preset?: string         // 'trending', 'popular', etc.
}
```

### 2. استخدام Component واحد لكل الصفحات

بدلاً من إنشاء 100+ ملف، نستخدم component واحد ذكي يتكيف حسب الـ props:

```typescript
// مثال: صفحة بحث الأفلام العربية
<Route 
  path="/arabic-movies/search" 
  element={<CategorySearchPage category="movies" subcategory="arabic" />} 
/>

// مثال: صفحة أفلام أكشن 2026
<Route 
  path="/movies/action/2026" 
  element={<CategorySearchPage category="movies" genre="action" year={2026} />} 
/>

// مثال: صفحة الدراما الكورية
<Route 
  path="/k-drama/search" 
  element={<CategorySearchPage category="series" subcategory="korean" />} 
/>
```
