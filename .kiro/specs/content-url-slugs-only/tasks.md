# خطة التنفيذ: روابط المحتوى بالـ Slugs فقط

## نظرة عامة

هذه الخطة تحول نظام الروابط من استخدام IDs كـ fallback إلى استخدام slugs نظيفة فقط. التنفيذ يتضمن ثلاث مراحل رئيسية:

1. بناء نظام توليد Slugs فريدة لجميع المحتوى
2. تحديث دوال توليد الروابط لاستخدام slugs فقط
3. إضافة دعم للروابط القديمة مع إعادة التوجيه
