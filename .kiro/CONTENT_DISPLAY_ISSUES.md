# 🚨 مشاكل عرض المحتوى - Content Display Issues

**تاريخ التقرير**: 2026-04-08
**الأولوية**: عالية جداً - CRITICAL

---

## المشاكل المكتشفة

### 1. الروابط بـ ID بدلاً من Slug
**المشكلة**: 
- روابط مثل `/watch/tv/tv-33238` خاطئة تماماً
- يجب أن تكون `/watch/tv/running-man` (slug فقط)

**السبب**: 
- المحتوى الجديد يُضاف بدون slugs
- الروتينج يقبل ID و slug معاً

**الحل المطلوب**:
- إجبار جميع الروابط على استخدام slug فقط
- إضافة slugs لجميع المحتوى الموجود
- تحديث منطق الروتينج

---

### 2. عرض العناوين خاطئ

**المشكلة الحالية**:
- يعرض عنوان واحد فقط (إنجليزي غالباً)
- مثال: "Running Man" فقط

**المطلوب**:
```
العنوان العربي: رانينج مان
العنوان الإنجليزي: Running Man  
العنوان الأصلي: 런닝맨
```

**الأولوية**: العربي أولاً، ثم الإنجليزي، ثم الأصلي

**الأعمدة في قاعدة البيانات**:
- `title_ar` - العنوان العربي
- `title_en` - العنوان الإنجليزي  
- `original_title` - العنوان الأصلي
- `title` - العنوان الافتراضي (يجب أن يكون عربي)

---

### 3. عرض الوصف خاطئ

**المشكلة الحالية**:
- يعرض الوصف الإنجليزي فقط
- لا يوجد وصف عربي

**المطلوب**:
- عرض الوصف العربي أولاً (إذا موجود)
- الوصف الإنجليزي كبديل
- ترجمة الأوصاف للعربية

**الأعمدة في قاعدة البيانات**:
- `overview` - الوصف (يجب أن يكون عربي)
- قد نحتاج `overview_ar` و `overview_en`

---

### 4. المواسم والحلقات

**المشكلة**:
- لا تظهر المواسم والحلقات بشكل صحيح
- مثال: Breaking Bad يعرض المواسم لكن بدون تفاصيل

**المطلوب**:
- عرض جميع المواسم مع عدد الحلقات
- عرض جميع الحلقات لكل موسم
- أسماء الحلقات بالعربي والإنجليزي

**الجداول المطلوبة**:
- `seasons` - جدول المواسم
- `episodes` - جدول الحلقات

---

### 5. المحتوى الجديد بدون Slugs

**المشكلة**:
- المحتوى الذي أُضيف مؤخراً ليس له slugs
- يستخدم IDs في الروابط

**السبب**:
- سكريبتات الإضافة لا تولد slugs
- لا يوجد validation على وجود slug

**الحل**:
1. إضافة slugs لجميع المحتوى الموجود
2. تحديث سكريبتات الإضافة لتوليد slugs
3. إضافة validation على مستوى قاعدة البيانات

---

## خطة الإصلاح

### المرحلة 1: إصلاح قاعدة البيانات
1. ✅ فحص الأعمدة الموجودة (تم)
2. ⏳ إضافة slugs للمحتوى بدون slugs
3. ⏳ التأكد من وجود title_ar و title_en لكل محتوى
4. ⏳ إضافة overview_ar إذا لم يكن موجود

### المرحلة 2: تحديث صفحة العرض (Watch.tsx)
1. ⏳ تحديث منطق جلب البيانات
2. ⏳ عرض العناوين الثلاثة (عربي، إنجليزي، أصلي)
3. ⏳ عرض الوصف العربي أولاً
4. ⏳ إصلاح عرض المواسم والحلقات

### المرحلة 3: تحديث الروتينج
1. ⏳ إزالة دعم ID من الروابط
2. ⏳ إجبار استخدام slug فقط
3. ⏳ إضافة redirect من ID إلى slug

### المرحلة 4: تحديث سكريبتات الإضافة
1. ⏳ تحديث جميع سكريبتات الإضافة لتوليد slugs
2. ⏳ إضافة validation على وجود slug
3. ⏳ إضافة ترجمة تلقائية للعناوين والأوصاف
